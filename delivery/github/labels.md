# Label Taxonomy

## Type
- `type:prd` — PRD authoring
- `type:adr` — Architecture decision
- `type:bug` — Bug
- `type:feature` — Feature
- `type:docs` — Documentation
- `type:incident` — Incident response
- `type:eval` — Eval / quality

## Priority
- `p0` — drop everything
- `p1` — this sprint
- `p2` — next sprint
- `p3` — backlog

## Status
- `status:blocked`
- `status:needs-review`
- `status:waiting-on-customer`
- `status:ready-for-merge`

## Domain
- `area:product`
- `area:engineering`
- `area:analytics`
- `area:delivery`
- `area:quality`
- `area:observability`
- `area:agents`
- `area:governance`

Apply at minimum: 1 type + 1 priority + 1 area.
